# 焼けたら死ぬ

週単位の収益を登録して、月別・年別のグラフで確認するブラウザアプリです。

## 使い方

1. Supabaseで新規プロジェクトを作成します。
2. SupabaseのSQL Editorで `supabase-schema.sql` を実行します。
3. GitHub Pagesでこのリポジトリを公開します。
4. 画面上部の同期設定に、Supabase URL、anon key、合言葉を入力します。

同じSupabase設定と同じ合言葉を使えば、PCとスマホで同じデータを同期できます。

## 注意

合言葉ログインは個人利用向けの簡易方式です。合言葉そのものは保存せず、合言葉から作ったIDを保存先として使います。
